Detailed Description

2D Line between two points.

Represents a line that goes through two points. Points are required to be integer. The architecture implies that there will be many segments lying on this line, all of which are stored in mems stack. Also note that line is "oriented" from Point A(ax, ay) to point B(bx, by). The orientation doesn't have any impact on performance, but might slightly modify the resulting structures.
