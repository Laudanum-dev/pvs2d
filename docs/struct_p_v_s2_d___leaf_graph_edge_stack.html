Detailed Description

Стэк ребер вершины графа листов.

Представляет стэк (связный список) ребер вершины графа листов, т.е. является списком связности (не путать со связным списком) вершины графа. Поскольку все соединения между листами идут через порталы, каждое ребро в графе представлено каким-то порталом.
